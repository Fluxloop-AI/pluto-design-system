## 사용

```tsx
import { ChatStepDot } from "@fluxloop-ai/pds-ui/components/chat-step-dot";

<ChatStepDot phase="running" />
<ChatStepDot phase="complete" size={8} />
```

`ChatBlock`·`ThinkingBlock`·`ToolCallCard` 공통 리더. 색은 `--pds-chat-dot-*` / `--pds-primary-normal` 에서 가져온다.
`running` 일 때만 ripple 애니메이션이 동심원으로 확산된다.

<ChatStepDotDemo />

## Props

| Prop | 타입 | 기본 | 설명 |
|---|---|---|---|
| `phase` | `ChatStatusPhase` | — | `pending \| running \| complete \| error \| requires-action` |
| `size` | `number` | `6` | px 단위 지름 |

## Phase → color 매핑

- **pending** — `--pds-chat-dot-idle` (= `--pds-label-assistive`)
- **running** — `--pds-primary-normal` + ripple
- **complete** — `--pds-chat-dot-done` (= `--pds-status-positive`)
- **error** — `--pds-chat-dot-error` (= `--pds-status-negative`)
- **requires-action** — `--pds-chat-dot-action` (= `--pds-status-cautionary`)
