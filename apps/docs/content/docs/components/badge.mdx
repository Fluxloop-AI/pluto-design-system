## Props

| Prop | 타입 | 기본 | 설명 |
|---|---|---|---|
| `variant` | `solid \| outlined` | `solid` | 배경 채움 / 외곽선만 |
| `size` | `xs \| sm \| md` | `xs` | 16 / 18 / 22 px 높이 |
| `color` | `neutral \| accent \| positive \| cautionary \| negative` | `neutral` | |
| `leadingContent` / `trailingContent` | `ReactNode` | - | 아이콘 등 |

## Montage 대비 변경점

- **이름**: Montage `ContentBadge` → **`Badge`** (제품명 변형 환원)
- **토큰 매핑**: `theme.semantic.*` → `var(--pds-fill-normal)`, `var(--pds-primary-normal)`, `var(--pds-status-*)`, `var(--pds-inverse-label)`
- **어두운 바리에이션**: outlined 는 배경 투명 + inset 1px shadow 테두리 (`color-mix` 로 color 별 35% alpha)
