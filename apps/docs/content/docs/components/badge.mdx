## Props

| Prop | 타입 | 기본 | 설명 |
|---|---|---|---|
| `variant` | `solid \| outlined` | `solid` | 배경 채움 / 외곽선만 |
| `size` | `xs \| sm \| md` | `xs` | 16 / 18 / 22 px 높이 |
| `color` | `neutral \| accent \| positive \| cautionary \| negative` | `neutral` | |
| `leadingContent` / `trailingContent` | `ReactNode` | - | 아이콘 등 |
