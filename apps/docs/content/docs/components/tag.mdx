```tsx
import { Tag } from "@fluxloop-ai/pds-ui/components/tag";

<Tag size="sm" pressed onClick={() => {}}>필터</Tag>
<Tag size="sm" variant="outlined" onRemove={() => {}}>태그</Tag>
```

## Props

| Prop | 타입 | 기본 | 설명 |
|---|---|---|---|
| `size` | `xs \| sm \| md \| lg` | `md` | |
| `variant` | `solid \| outlined` | `solid` | |
| `pressed` | `boolean` | - | 지정 시 `aria-pressed` 연결 |
| `leadingContent` / `trailingContent` | `ReactNode` | - | |
| `onRemove` | `() => void` | - | 지정 시 trailing 자리에 × 아이콘 |
| `asChild` | `boolean` | - | 루트를 `<a>` 등으로 교체 |

## Montage 대비 변경점

- **이름**: Montage `Chip` → **`Tag`** (제품명 변형 환원)
- **폴리모픽**: `asChild` 지원 (`@radix-ui/react-slot`)
- **pressed**: `aria-pressed` + `data-pressed` 양방향 — 스타일은 `data-pressed=true` 선택자 사용
