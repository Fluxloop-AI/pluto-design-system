## 사용

```tsx
import { ToolCodeBlock } from "@fluxloop-ai/pds-ui/components/tool-code-block";

<ToolCodeBlock code={json} language="json" maxHeight={160} />

// shiki / hljs 주입
<ToolCodeBlock
  code={src}
  language="json"
  renderCode={(c, lang) => <ShikiCode code={c} lang={lang} />}
/>
```

Shiki 같은 헤비 의존성은 PDS 에 포함하지 않는다. `renderCode` slot 을 통해 앱 측에서 lazy-load 한다.
미제공 시 plain `<pre><code>` fallback 렌더.

## Props

| Prop | 타입 | 기본 | 설명 |
|---|---|---|---|
| `code` | `string` | — | 코드 본문 |
| `language` | `"json" \| "text"` | `"json"` | 언어 라벨 |
| `tone` | `"default" \| "caution" \| "negative"` | `"default"` | 텍스트 색 |
| `maxHeight` | `number` | — | 접힘 시 최대 높이 (px) |
| `expanded` | `boolean` | `false` | true 면 maxHeight 무시 |
| `renderCode` | `(code, lang) => ReactNode` | — | 하이라이터 slot |
