## 사용

```tsx
import { Avatar } from "@fluxloop-ai/pds-ui/components/avatar";

<Avatar size="lg" variant="person" src="/me.jpg" alt="me" />
<Avatar size="md" variant="company" />
```

## Size

8 단계 — `xs`(20) · `sm`(24) · `md`(32) · `lg`(40) · `xl`(48) · `2xl`(56) · `3xl`(64) · `4xl`(80).

<AvatarSizeDemo />

## Variant

- **person** — 원형 (9999px)
- **company / academy** — 라운드 스퀘어 (size × ~0.125)

fallback 아이콘은 `@fluxloop-ai/pds-icons/lucide` 의 `User` / `Building2` / `GraduationCap`.

<AvatarVariantDemo />

## Props

| Prop | 타입 | 기본 | 설명 |
|---|---|---|---|
| `size` | `xs \| sm \| md \| lg \| xl \| 2xl \| 3xl \| 4xl` | `md` | 8단계 스케일 |
| `variant` | `person \| company \| academy` | `person` | 모양 + 기본 fallback 아이콘 |
| `src` | `string` | — | 이미지 URL. 없으면 바로 fallback 렌더 |
| `alt` | `string` | `""` | 이미지 alt |
| `fallback` | `ReactNode` | 자동 | 커스텀 fallback (아이콘/이니셜 등) |

## Montage 대비 변경점

- **headless**: 자체 loading state → `@radix-ui/react-avatar` 의 `Image` / `Fallback` 이 자동 처리
- **size 확장**: 5단계 → **8단계** (xs 20 / 4xl 80 추가, 나머지는 Montage 유지)
- **fallback 아이콘**: `@pluto/wds-icon` → `@fluxloop-ai/pds-icons/lucide`
- **토큰 치환**: `theme.semantic.*` → `var(--pds-*)`
- **responsive props 제거**: `xs/sm/md/lg/xl` breakpoint 오버라이드 제거 (Phase 1 범위 밖, 필요 시 Phase 5+)
