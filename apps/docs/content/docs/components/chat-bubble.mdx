## 사용

```tsx
import { ChatBubble } from "@fluxloop-ai/pds-ui/components/chat-bubble";

<ChatBubble content="안녕하세요" />
<ChatBubble content={[
  { type: "text", text: "아래 이미지 참고" },
  { type: "image", source: { type: "url", url: "/foo.png" } },
]} />
```

`content` 는 문자열 또는 Anthropic 콘텐츠 블록 배열. 이미지 로드 실패 시 자동으로 fallback 박스로 대체.
어시스턴트 메시지 본문은 풍선 없이 `ChatThread` 가 `renderMarkdown` slot 으로 직접 그린다.

## Props

| Prop | 타입 | 기본 | 설명 |
|---|---|---|---|
| `content` | `string \| ContentBlock[]` | — | 메시지 본문 |
| `role` | `"user"` | `"user"` | 현재 user 풍선만 지원 |
| `actions` | `ReactNode` | — | 풍선 아래 action bar 슬롯 (copy/send-to-note 등) |
| `className` | `string` | — | 루트 클래스 |
