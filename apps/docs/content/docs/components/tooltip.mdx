## 사용

```tsx
import {
  Tooltip,
  TooltipProvider,
  TooltipTrigger,
  TooltipContent,
} from "@fluxloop-ai/pds-ui/components/tooltip";

<TooltipProvider delayDuration={200}>
  <Tooltip>
    <TooltipTrigger asChild>
      <Button>Hover me</Button>
    </TooltipTrigger>
    <TooltipContent shortcut="⌘K">명령 팔레트</TooltipContent>
  </Tooltip>
</TooltipProvider>
```

## Props

### `<Tooltip />`

| Prop | 타입 | 기본 | 설명 |
|---|---|---|---|
| `mode` | `hover \| always \| click` | `hover` | `always` 는 항상 열림, `click` 은 controlled toggle 권장 |
| `open` / `defaultOpen` / `onOpenChange` | Radix | — | 표준 controlled API |
| `delayDuration` | `number` | `700` | Provider 값 상속 |

### `<TooltipContent />`

| Prop | 타입 | 기본 | 설명 |
|---|---|---|---|
| `size` | `sm \| md` | `md` | 패딩 + 폰트 크기 |
| `shortcut` | `ReactNode` | — | 오른쪽 shortcut slot (⌘K 등) |
| `arrow` | `boolean` | `false` | 화살표 표시 |
| `side` / `align` / `sideOffset` | Radix | — | 표준 positioning |

## Montage 대비 변경점

- **headless**: 자체 popper → `@radix-ui/react-tooltip` (a11y + positioning)
- **mode 간소화**: `hover` (기본) + `always` (open=true) + `click` (controlled + `disableHoverableContent`)
- **shortcut slot 유지**: Montage 와 동일한 slot 구조
- **토큰 치환**: `theme.semantic.inverse.*` → `var(--pds-inverse-*)`, z-index `--pds-z-tooltip` (1500)
