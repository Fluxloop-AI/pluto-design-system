## 사용

```tsx
import { Tabs, TabsList, TabsTrigger, TabsContent } from "@fluxloop-ai/pds-ui/components/tabs";

<Tabs defaultValue="overview">
  <TabsList>
    <TabsTrigger value="overview">개요</TabsTrigger>
    <TabsTrigger value="members">멤버</TabsTrigger>
  </TabsList>
  <TabsContent value="overview">...</TabsContent>
  <TabsContent value="members">...</TabsContent>
</Tabs>
```

## Props

### Tabs

| Prop | 타입 | 기본 | 설명 |
|---|---|---|---|
| `value` / `defaultValue` / `onValueChange` | Radix 표준 | - | |
| `orientation` | `horizontal \| vertical` | `horizontal` | |
| `activationMode` | `automatic \| manual` | `automatic` | Radix 기본 |
| `size` | `sm \| md \| lg` | `md` | context 로 자식에 전파 |
| `resize` | `hug \| fill` | `hug` | fill 이면 트리거가 폭 나눔 |

### TabsTrigger

| Prop | 타입 | 기본 | 설명 |
|---|---|---|---|
| `value` | `string` | - | 필수 |
| `disabled` | `boolean` | - | |
