## 사용

```tsx
import { ThinkingBlock } from "@fluxloop-ai/pds-ui/components/thinking-block";
import ReactMarkdown from "react-markdown";

<ThinkingBlock
  thinking={reasoning}
  statusPhase={phase}
  renderMarkdown={(text) => <ReactMarkdown>{text}</ReactMarkdown>}
/>
```

- `running` 진입 시 자동 펼침 + `pds-animate-glow` 활성.
- `running → complete` 전이 시 duration 계산 후 1초 뒤 자동 접힘.
- 수동 토글 이후에는 auto-close 가 다시 발동하지 않음.
- `renderMarkdown` 미제공 시 `<pre>` fallback.

## Props

| Prop | 타입 | 기본 | 설명 |
|---|---|---|---|
| `thinking` | `string` | — | 원본 reasoning 텍스트 |
| `duration` | `number` | — | 외부에서 계산한 초 단위 지속시간 (있으면 우선) |
| `redacted` | `boolean` | `false` | 안전 필터로 숨겨진 경우 |
| `statusPhase` | `ChatStatusPhase` | `"complete"` | 도트 색 + 자동 열기/접힘 트리거 |
| `renderMarkdown` | `(text: string) => ReactNode` | — | markdown slot |
