## 사용

```tsx
import { ChatTabBar, type ChatTab } from "@fluxloop-ai/pds-ui/components/chat-tab-bar";

<ChatTabBar
  tabs={tabs}
  activeId={activeId}
  onSwitch={(id) => store.switchConversation(id)}
  onClose={(id) => store.closeConversation(id)}
/>
```

- 완전 controlled. 스토어 주입 없음.
- `active` / hover 상태에서만 close 버튼 노출.
- 가로 스크롤 되며 스크롤바는 숨김. 가로 자리가 부족하면 wheel 로 이동.

## Props

| Prop | 타입 | 기본 | 설명 |
|---|---|---|---|
| `tabs` | `ChatTab[]` | — | `{ id, title }` 배열 |
| `activeId` | `string \| null` | — | 현재 활성 탭 |
| `onSwitch` | `(id: string) => void` | — | 탭 클릭 |
| `onClose` | `(id: string) => void` | — | 닫기 클릭 |
