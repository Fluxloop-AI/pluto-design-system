**IconButton — `subtle` variant 추가.**

평소 `--pds-label-alternative`(흐림) → hover 시 `--pds-label-normal`(또렷) + `--pds-fill-normal` 배경. 헤더·툴바·sidebar 액션 슬롯처럼 본 콘텐츠 옆에서 시선을 안 뺏어야 하는 보조 액션용. 기존 `normal` variant는 평소부터 진한 톤이라 정렬되지 않음.

데모 적용처: `chat-composer`(첨부·음성), `sidebar-list`(헤더 액션 3개), `section-header`(headingContent / trailingContent).
